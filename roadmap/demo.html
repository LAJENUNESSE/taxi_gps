深圳出租车GPS数据分析系统 · 演示入口

四个基于百度地图 JavaScript API 的交互可视化页面，集中展示 OD 流向、轨迹回放、上客点热力与路网拥堵。

访问说明

所有交互页面基于百度地图 JavaScript API，受浏览器安全策略限制，不能使用 file:// 协议打开。请在项目根目录启动 HTTP 服务器后访问：

python -m http.server 8080

然后浏览器打开 http://localhost:8080/roadmap/demo.html 或直接访问下列各页面。

演示页面

OD 流向图

将 486K 出行对按 ~1.5 km 网格聚合，取 Top 300 流向线，颜色与粗细反映流量大小。

打开页面

路径：output/figures/baidu_flow_map.html

轨迹回放

100 辆代表性车辆的轨迹回放，速度渐变折线、上下客标记与实时信息面板。

打开页面

路径：output/figures/trajectory_viewer.html

交互热力图

上客点按小时切换的交互热力图，~100 m 网格聚合，时间滑块 0–23h。

打开页面

路径：output/figures/interactive_heatmap.html

路网拥堵

深圳路网按平均速度着色（绿畅通/黄缓行/红拥堵），基于 GPS 路段速度统计。

打开页面

路径：output/figures/road_congestion.html

请使用 python -m http.server 8080 启动后访问 http://localhost:8080/output/figures/xxx.html。
