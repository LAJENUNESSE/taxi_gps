专业实训 V：出租车 GPS 数据分析与可视化

本项目围绕出租车 GPS 数据完成清洗、OD 提取、缓存构建、地图展示、热力图、统计分析、路网校正、拥堵道路展示、ETA 预测和系统集成。

系统调用主线

数据读取与清洗 -> OD 提取 -> 车辆缓存 / 分钟缓存 / OD 缓存 -> 轨迹查询 / 车辆位置查询 -> 热力图 / 统计分析 -> 路网校正 / HMM 匹配 -> 拥堵道路颜色 -> ETA 预测 -> 系统集成
